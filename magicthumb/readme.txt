INSTALLATION:

NOTE: To manually install Magic Thumb please read 'readme_manual.txt'.

1. Unzip the contents of the zip file and upload them to your Magento directory.

2. Go to http://your.site.url/magicthumb/ in your browser and follow the instructions.

3. Open the 'app/etc/magictoolbox/magicthumb.settings.ini' file and configure Magic Thumb. If you want different configurations for different themes you can place a separate 'magicthumb.settings.ini' file into the 'app/design/frontend/[your_interface]/[your_theme]/' folder.

4. You've now installed the demo version of Magic Thumb!

5. To upgrade, buy Magic Thumb and overwrite the magicthumb.js file with the same file from the full version.
